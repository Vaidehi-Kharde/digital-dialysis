# 📱 Remote Peritoneal Dialysis Management System

A mobile-based healthcare solution designed to enable **remote peritoneal dialysis management** for patients and doctors. This project is actively deployed and being used at **AIMS Nagpur**, helping bridge the gap between hospital care and at-home treatment.

---

## 🚀 Overview

This application streamlines the entire peritoneal dialysis workflow — from patient onboarding to session completion — ensuring **safe, structured, and doctor-verified remote dialysis**.

Built using **Flutter (Dart)** for cross-platform mobile development and powered by a **Vercel-hosted backend**, the system ensures scalability, accessibility, and real-time coordination between patients and healthcare providers.

---

## ✨ Key Features

### 👤 Patient Management

* Patient registration via mobile app
* Secure onboarding and profile creation
* Assignment of patients to doctors

### 🏥 Dialysis Material Workflow

* Patients collect dialysis materials from the hospital
* Doctors verify materials before session approval
* Ensures correctness and safety before procedure

### 📅 Smart Scheduling System

* Doctors create and assign dialysis sessions
* Custom scheduling tailored to patient needs
* Session tracking and management

### 🩺 Pre-Dialysis Wellness Check

* Mandatory health form before each session
* Captures vital pre-procedure indicators
* Prevents unsafe dialysis attempts

### ✅ Session Verification & Monitoring

* Dialysis session validation workflow
* Doctor-side verification for compliance
* Structured and traceable treatment lifecycle

## 📦 Installation & Setup

### Prerequisites

* Flutter SDK installed
* Android device or emulator

### Steps

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd <project-folder>

# Install dependencies
flutter pub get

# Build APK
flutter build apk
```

### 📲 Run the Application

* Transfer the generated APK to your Android device
* Install the APK
* Launch the app and start using the system
